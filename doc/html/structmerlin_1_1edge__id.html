Detailed Description

Edge in the graph.

EdgeID represents the edge (i->j) with index idx. The reversed index ridx represents the edge (j->i). Blank (or missing) edges can sometimes appear, and they are represented by EdgeID::NO_EDGE (for example, if edge (i,j) is requested and does not exist).
