Detailed Description

Weighted Mini-Buckets (WMB)

Tasks supported: PR, MAR, MAP, MMAP

WMB generalizes the classical MBE by replacing the sum operator with a weighted sum operator, and thus it uses Holder's inequality to derive an upper bound on the log partition function, MAP or Marginal MAP value. WMB also uses an iterative cost-shifting scheme that matches marginals (weighted or max) in order to tighten the upper-bound. Tightening is not guaranteed in general, but it typically happens in practice.
