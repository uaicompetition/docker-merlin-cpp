Detailed Description

Bucket Tree Elimination (BTE)

Tasks supported: PR, MAR, MAP, MMAP

Bucket Tree Elimination is an exact inference algorithm that supports the main inference tasks in graphical models (PR, MAR, MAP, MMAP). It belongs to the class on join-tree elimination algorithms and is driven by a bucket tree structure. Its complexity is time and space exponential in the induced width. Note that this is meant to be a reference implementation and therefore is applicable to relatively small models having relatively small induced widths.
