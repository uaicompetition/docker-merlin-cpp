Detailed Description

Join-Graph Linear Programming (ie, Weighted mini-buckets for MAP inference)

Tasks supported: MAP

JGLP is a specialization of the weighted mini-bucket algorithm to the MAP inference task. It builds a join graph by running the mini-bucket algorithm schemetically, and then it connects the mini-buckets residing in the same bucket. JGLP shifts costs along the edges of the join graph by matching the max marginals of any two clusters connected by an edge. In practice, the algorithm typically tightens the MAP upper bound, however this is not a guarantee. It keeps track of the tightest upper bound found and its corresponding assignment.
